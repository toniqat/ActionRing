| `action-sequence.svg` | Action sequence group |
| `action-shell.svg` | Shell command |

| `action-system_action.svg` | System action (volume, brightness, etc.) |
| `action-text.svg` | Text input / clipboard |
| `action-toast.svg` | Toast notification |
| `action-variable.svg` | Variable set/get |
| `action-wait.svg` | Wait / delay |
